Description:

Goblet: Death and Rebirth is a top-down roguelite that gathers inpiration from both The Binding of Isaac: Rebirth and Goblet Grotto. In its final form the game is intended to include the following features:

  • Randomly generated levels, so that every time the game starts the player gets a unique experience

  • A simple combat system where the main character can attack in 4 or 8 different directions

  • Many different enemies with different attack patterns

How to play:

Use the arrow keys to move the main character.
Attack with Z. You will attack in the direction the player is facing, however the attack direction will remain the same even if you change player direction once you have started the attack.
